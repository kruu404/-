# 第二章 进程管理

## 1. 进程创建

### 创建步骤
- ① 申请空白PCB  
- ② 为新进程分配资源  
- ③ 初始化进程控制模块  
- ④ 将新进程插入就绪队列  

### 触发事件
- ① 用户登录后  
- ② 作业调度  
- ③ 操作系统提供服务  
- ④ 进程本身应用请求（如启动程序执行时）
